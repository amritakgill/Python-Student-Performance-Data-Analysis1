# Student Lifestyle & GPA Analysis

## Overview
This project analyzes how college students’ daily habits relate to academic performance. The analysis focuses on lifestyle factors such as study time, sleep, physical activity, social time, extracurricular involvement, stress level, and GPA.

## Research Question
How do lifestyle habits, especially physical activity, relate to GPA among college students?

## Dataset
The dataset includes information from 2,000 college students between the ages of 18 and 24.

Key variables include:

- Study hours
- Sleep hours
- Physical activity
- Social hours
- Extracurricular hours
- Stress level
- GPA

## Methods
This project used Python to:

- Clean and organize the dataset
- Rename variables for readability
- Convert categorical variables into usable formats
- Remove invalid observations
- Explore trends in student lifestyle habits
- Visualize relationships between lifestyle factors and GPA

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Statsmodels

## Key Findings
The project explored how different student habits may be associated with GPA. Study time, sleep, physical activity, and stress were examined as possible factors related to academic performance.
